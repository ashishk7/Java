# Problem Description
Given a number A. Return 1 if A is prime and return 0 if not.

>[!Note]
>The value of A can cross the range of Integer.


## Problem Constraints
```java
1 <= A <= 109
```

## Input Format
```java
The first argument is a single integer A.
```

## Output Format
```java
Return 1 if A is prime else return 0.
```

## Example Input
### Input 1:
```java
A = 5
```
### Input 2:
```java
A = 10
```

## Example Output
### Output 1:
```java
1
```

### Output 2:
```java
0
```

## Example Explanation
### Explanation 1:
```java
5 is a prime number.
```

### Explanation 2:
```java
10 is not a prime number.
```
