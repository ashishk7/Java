# Problem Description
You are given an array A of N integers. Return the count of elements with frequncy 1 in the given array.


## Problem Constraints
```java
1 <= N <= 10^5
1 <= A[i] <= 10^9
```

## Input Format
First argument `A` is an array of integers.


## Output Format
Return an integer.


## Example Input
### Input 1:
```java
A = [3, 4, 3, 6, 6]
```
### Input 2:
```java
A = [3, 3, 3, 9, 0, 1, 0]
```

## Example Output
### Output 1:
```java
1
```
### Output 2:
```java
2
```


## Example Explanation
### Explanation 1:
```java
Only the element 4 has a frequency 1.
```

### Explanation 2:
```java
The elements 9 and 1 has a frequncy 1.
```
